# 人设 + 商业定位 API

AI 驱动的博主人设分析和商业定位推荐。

## 端点列表

### 人设 (Persona) — `/api/v1/personas`

| 方法 | 路径 | 鉴权 | 说明 |
|---|---|---|---|
| `POST` | `/personas/scan` | JWT | 人设扫描 |
| `GET` | `/personas/me` | JWT | 我的人设档案 |
| `GET` | `/personas/archetypes` | 无 | 所有人设原型列表 |

### 商业定位 (Positioning) — `/api/v1/positionings`

| 方法 | 路径 | 鉴权 | 说明 |
|---|---|---|---|
| `POST` | `/positionings/scan` | JWT | 商业定位扫描 |
| `GET` | `/positionings/me` | JWT | 我的定位档案 |

---

## 人设扫描

```
POST /api/v1/personas/scan
```

基于视频内容 + 用户自述，AI 反向工程博主的人设原型。

### 请求体

```json
{
  "sample_video_urls": [
    "https://v.douyin.com/xxxxx/",
    "https://v.douyin.com/yyyyy/"
  ],
  "description": "我想做一个有态度、有点毒舌但又真诚的美食测评博主"
}
```

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `sample_video_urls` | `string[]` | 否 | 代表性视频链接列表，用于 AI 提取内容风格 |
| `description` | `string` | 否 | 用户对自身定位的文字描述 |

### 响应

```json
{
  "code": "OK",
  "message": "success",
  "data": {
    "id": 1,
    "primary_archetype": "气场型",
    "sub_archetype": "反差大佬",
    "contrast_point": "看似温吞实则毒舌精准，形成强烈反差",
    "self_tags": {
      "关键词": ["美食", "测评", "毒舌"],
      "调性": "犀利 + 专业"
    },
    "audience_tags": {
      "粉丝画像": ["18-30岁", "一二线城市", "喜欢尝鲜"],
      "核心需求": ["避雷指南", "种草参考", "娱乐解压"]
    },
    "scores": {
      "气场力": 82,
      "干货力": 75,
      "共情力": 60,
      "趣味力": 70,
      "商业力": 55
    },
    "consistency_score": 78,
    "canvas": {
      "品牌定位": "毒舌专业的美食判官",
      "核心价值": "帮年轻人避开网红餐厅的坑",
      "目标受众": "一二线城市 18-30岁 美食爱好者"
    },
    "diagnosis": {
      "现状": "已有差异化人设雏形，但内容调性不够统一",
      "建议": "保持毒舌风格一致性，增加专业资格证书背书"
    },
    "drift_alert": false,
    "created_at": "2026-05-20T10:30:00Z"
  }
}
```

### 四项人设核心指标

| 指标 | 说明 |
|---|---|
| `气场力` | 个人魅力、权威感、辨识度 |
| `干货力` | 信息价值、专业深度、可操作性 |
| `共情力` | 情感连接、粉丝认同、社区归属 |
| `趣味力` | 娱乐性、传播力、内容趣味度 |
| `商业力` | 变现能力、商业价值、品牌合作潜力 |

## 我的人设档案

```
GET /api/v1/personas/me
```

获取当前用户最新的人设扫描结果。如未扫描过，`data` 为 `null`。

## 人设原型列表

```
GET /api/v1/personas/archetypes
```

无需鉴权。返回所有可用的人设原型体系。

### 响应

```json
{
  "code": "OK",
  "message": "success",
  "data": {
    "气场型": ["行业权威", "反差大佬", "凶猛敢说"],
    "干货型": ["教学派", "拆解派", "资源派"],
    "共情型": ["闺蜜型", "治愈系", "共同体"],
    "趣味型": ["段子手", "鬼马少年", "反差萌"]
  }
}
```

### 原型体系说明

| 大类 | 子类型 | 典型特征 |
|---|---|---|
| **气场型** | 行业权威 | X 年经验、资格证书背书、权威语调 |
| | 反差大佬 | 外表随和 + 专业硬核的反差效果 |
| | 凶猛敢说 | 犀利观点、不怕得罪人、话题性强 |
| **干货型** | 教学派 | 步骤清晰、手把手教、可复现 |
| | 拆解派 | 深度分析、逻辑严密、数据说话 |
| | 资源派 | 工具推荐、资源合集、省时省力 |
| **共情型** | 闺蜜型 | 亲切私密、像跟朋友聊天 |
| | 治愈系 | 温暖放松、情绪价值、解压 |
| | 共同体 | 身份认同、圈层归属、价值观共鸣 |
| **趣味型** | 段子手 | 笑点密集、轻松娱乐 |
| | 鬼马少年 | 活泼搞怪、青春感、潮流 |
| | 反差萌 | 严肃外表下的可爱行为 |

## 商业定位扫描

```
POST /api/v1/positionings/scan
```

基于博主的赛道、粉丝量、现状，AI 推荐最优商业定位和变现路径。

### 请求体

```json
{
  "description": "我现在做美食内容，粉丝5万，想知道怎么变现"
}
```

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `description` | `string` | 否 | 补充说明（AI 也会读取你的用户 profile） |

### 响应

```json
{
  "code": "OK",
  "message": "success",
  "data": {
    "id": 1,
    "scores": {
      "内容广告": 75,
      "知识付费": 60,
      "电商带货": 80,
      "品牌合作": 55,
      "私域变现": 45
    },
    "monetization_paths": {
      "推荐路径": "电商带货——美食赛道天然适合好物推荐",
      "备选路径": ["内容广告", "知识付费——课程化你的美食方法论"]
    },
    "recommended_archetype": "干货型",
    "recommended_routes": {
      "3个月": "建立信任人设，开始软植入带货",
      "6个月": "推出自有品牌或联名产品",
      "12个月": "建立美食社群 + 培训体系"
    },
    "avoid_routes": {
      "不建议": ["硬广轰炸——美食赛道硬广转化率低", "过度依赖单一平台"]
    },
    "roadmap_12m": {
      "Q1": "打磨内容 + 建立信任 + 100个种子用户",
      "Q2": "首次带货测试 + 收集反馈",
      "Q3": "规模化内容生产 + 多渠道分发",
      "Q4": "自有品牌/课程产品上线"
    },
    "risk_level": 2,
    "canvas_bmc": {
      "价值主张": "帮忙碌的年轻人找到真正好吃的外卖",
      "客户关系": "基于信任的推荐关系",
      "收入来源": "带货佣金 + 品牌合作 + 课程",
      "关键资源": "精准口味描述 + 供应链资源"
    },
    "created_at": "2026-05-20T10:30:00Z"
  }
}
```

### 定位评分说明

| 评分维度 | 说明 |
|---|---|
| `内容广告` | 在内容中自然地植入广告的适合度 |
| `知识付费` | 将知识系统化出售的适合度 |
| `电商带货` | 通过直播/视频带货的适合度 |
| `品牌合作` | 与品牌建立长期合作的适合度 |
| `私域变现` | 将粉丝导入私域（微信/社群）变现的适合度 |

### 风险等级

| risk_level | 含义 |
|---|---|
| 1 | 低风险 — 赛道成熟，变现路径清晰 |
| 2 | 中低风险 — 需注意节奏和内容质量 |
| 3 | 中风险 — 需谨慎选择变现方式 |
| 4 | 中高风险 — 存在平台政策或竞争压力 |
| 5 | 高风险 — 建议重新评估赛道和定位 |
