## 9.3 定价心理学（为什么是 99 和 499）

### PRO 99 的心理设计

| 维度 | 99 元的心理锚点 |
|---|---|
| 价位锚 | "百元以内" → 不需要老婆批准 |
| 替代锚 | "一杯星巴克 = 4 杯奶茶 = 半部电影票" |
| 频率锚 | "365 天 / 4 = 25 元一周" |
| 收益锚 | "改对一条视频涨 1000 粉 = 回本" |
| 时间锚 | "买课 99 还看不完，买这个每天都用" |

### MAX 499 的心理设计

| 维度 | 499 元的心理锚点 |
|---|---|
| 价位锚 | "5 张毛爷爷" → 商务人士标准月度小决策 |
| 替代锚 | "一次理发 + 一顿火锅 + 一次按摩" |
| 收益锚 | "拍一条广告分 5000，多接一条就回本 10 倍" |
| 对照锚 | "找私教 8000 一月 = 16 个月 MAX" |
| 时间锚 | "一年 4999 = 13.7 元/天" |

### 关键禁忌
- ❌ 不要定 88、188、299 这类"民俗折扣感"价位
- ❌ 不要做"前 100 名 9.9"这类廉价感引流
- ❌ 不要"原价 999 限时 99"这类对账感
- ✅ 99 和 499 直接定价，不打折感，让客户感受到"专业产品"

## 9.4 续费策略

### 续费的 3 大底层杠杆

#### 杠杆 1：可视化进步（情绪杠杆）
- PRO 每月推送差距曲线
- MAX 每月推送差距进度条 + 等级
- 数字一旦在上升，续费决策几乎是反射

#### 杠杆 2：沉没成本（理性杠杆）
- 终身档案：付费越久越厚
- 历史模板下载：付费越久积累越多
- 顾问关系（MAX）：换地方要重新解释一遍

#### 杠杆 3：损失厌恶（防御杠杆）
- "断订后 30 天内可恢复，否则差距清零"
- "续费可保留 6 个月历史档案 / 终身档案"
- "MAX 断订需重新排队等顾问名额"

### 续费机制设计

| 触发节点 | 动作 | 工具 |
|---|---|---|
| 续费日 -14 天 | 顾问主动联系（MAX） | 工单系统 |
| 续费日 -7 天 | 推送进度报告 + 续费按钮 | 小程序订阅消息 |
| 续费日 -3 天 | 推送限时优惠 | App 消息 + 短信 |
| 续费日 -1 天 | 顾问 1v1（仅 MAX） | 电话 |
| 续费日当天 | 自动扣款（已签自动续费） | 微信支付 |
| 续费日 +1 天 | 已断订挽留首响（顾问） | 电话 + 微信 |
| 续费日 +7 天 | 流失访谈邀请 | 问卷 + 30 元红包 |
| 续费日 +30 天 | 流失客户名单转召回池 | 月度内容推送 |

### 续费率目标

| 周期 | PRO 续费率 | MAX 续费率 |
|---|---|---|
| 月度（M+1） | 70% | 85% |
| 季度（M+3） | 50% | 70% |
| 年度（M+12） | 30% | 50% |
| 2 年（M+24） | 15% | 30% |

## 9.7 合规框架

### 公司主体与资质
- 注册有限责任公司（建议杭州/广州/深圳）
- 经营范围包括：互联网信息服务、技术开发、技术咨询、市场营销策划
- 必须办理：营业执照、税务登记、ICP 备案、公安网络备案
- 视情况办理：增值电信业务经营许可证（涉及付费 SaaS）
- 高频次直播：申请网络文化经营许可证

### 服务协议三件套
1. **用户注册协议**（小程序勾选）
2. **服务订阅协议**（付费时签订电子版）
3. **MAX 顾问陪跑协议**（MAX 客户额外签订）

### 数据合规
- 隐私政策（注册前必读）
- 用户数据出境申报（如有海外业务）
- 数据处理者备案
- 重要数据安全评估

### 营销红线
- ❌ "包爆款"、"必涨粉"、"保证收益"等绝对化用语
- ❌ "刷量"、"控评"、"刷粉"等违规承诺
- ❌ 虚构客户案例、伪造数据
- ❌ 未经客户授权对外公开账号信息
- ❌ 不实承诺"24 小时见效"

### 财税合规
- 对公收款（不收个人现金）
- 月度增值税申报
- 季度企业所得税申报
- 员工工资代扣代缴
- 第三方账户审计（年度）

## 9.8 商业模式画布（BMC）

| 模块 | 内容 |
|---|---|
| **客户细分** | PRO：0-10 万粉新手；MAX：10 万+ 中腰部；企业：MCN/品牌矩阵 |
| **价值主张** | 影像式诊断 + 头部对标 + 终身档案 + 合规变现 |
| **渠道** | 抖音/视频号自有内容、小红书/公众号、PRO→MAX 升级、加盟商 |
| **客户关系** | PRO 自助 + 社群；MAX 1v1 顾问 + 季度战略会 |
| **收入来源** | 订阅（75%）+ 加购（15%）+ 生态（10%） |
| **关键资源** | 头部对标数据库、CT 诊断模型、顾问团队、案例库 |
| **关键业务** | 数据采集、AI 诊断、人工陪跑、内容输出 |
| **关键合作** | 第三方数据商、AI 模型商、加盟商、平台政策对接人 |
| **成本结构** | 人力（45%）+ 投放（30%）+ AI/服务器（10%）+ 内容生产（10%）+ 行政（5%） |
