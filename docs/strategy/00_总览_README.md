> 文档版本：v1.0 · 2026-05-20

> 适用范围：商业体系搭建、团队 SOP、外部投放、招商加盟

---

## 一句话战略

**用 CT 影像式诊断方法论，把"短视频改稿"做成可订阅、可复购、可量化、可规模化的 SaaS+陪跑产品。**

- PRO（99/月）解决"我不知道怎么改"——AI 月度体检 + 模板套用
- MAX（499/月）解决"我不知道差在哪、怎么追上去"——十大头部对标 + 人工战略陪跑 + 终身成长档案

## 双产品定位一图对照

| 维度 | PRO（99/月） | MAX（499/月） |
|---|---|---|
| 目标人群 | 0–10 万粉的普通博主、新手、个体副业 | 10 万+ 粉的中腰部/准头部、品牌主、矩阵主理人 |
| 核心承诺 | 每月一次完整 AI 体检，知道改哪里、给模板 | 每月对标十大头部，量化差距 + 人工战略校准 |

| 诊断深度 | 单条 6 维 CT（表层/内核/视听/人设/数据/变现） | 6 维 CT + 六大指标对标 + 等级晋升 + 行业趋势 |
| 对标参照 | 同赛道头部样本均值（库存对标） | 客户专属赛道 TOP10 永久对标池（动态更新） |

| 锁客逻辑 | 模板订阅 + 月度成长曲线 | 终身档案 + 等级晋升 + 头部资源对接 |
| 边际成本 | 极低（≈ 100% AI） | 中（人工顾问占用） |
| 毛利率目标 | 85%+ | 65%+ |
| 单客年 ARPU | 1,188（年付折扣 ~999） | 5,988（年付折扣 ~4,999） |

## 落地优先级（如果只能做 5 件事）

1. **MVP 上线 PRO 99/月 + 免费 3 次扫描**（30 天内）：AI 诊断引擎 + 微信小程序入口 + 单次 19 / PRO 99 双产品
2. **四端骨架同步搭建**（30 天内并行）：C 端 + 顾问端 + 运营端 + 开放端，详见 14 章
3. **筛 30 名 MAX 种子客户**（30–60 天）：用人工顶 AI、跑通对标库与档案 SOP，沉淀 5 个成长案例
4. **建对标头部库 1.0 + 8 Agent 矩阵基础版**（60 天内）：每细分赛道固定 TOP10、4 大 AI 飞轮开始累积数据
5. **上线分享官体系 + 任务激励**（60–90 天）：把获客从"投流"变成"老客户拉新客户"
