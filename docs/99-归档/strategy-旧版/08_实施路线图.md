## 8.2 M0-M1：MVP 期（Day 0–30）

### 核心目标
跑通 PRO 99 完整闭环：从注册到出报告到续费提醒

### 关键里程碑
- ✅ Day 7：CT 诊断引擎 v0.1 跑通（单条视频出 6 维报告）
- ✅ Day 14：微信小程序 + 支付链路上线
- ✅ Day 21：5 个内测客户付费，反馈完整一轮
- ✅ Day 30：正式公开发售，首月 30 单 PRO

### 周任务拆解

**Week 1（Day 1-7）：技术地基**
| 任务 | 负责人 | 交付物 |
|---|---|---|
| 服务器、域名、SSL 申请 | 技术 | 阿里云账号 + 域名 |
| 后端框架搭建（FastAPI） | 技术 | API 骨架 |
| 数据库设计（用户/订阅/诊断/档案） | 技术 | ER 图 + 建表 SQL |
| CT 诊断 Prompt v0.1 | 内容 | 6 维 prompt + 5 个 few-shot |
| 头部账号库初始化（5 个赛道） | 运营 | 50 个头部账号画像 |

**Week 2（Day 8-14）：MVP 上线**
| 任务 | 负责人 | 交付物 |
|---|---|---|
| 视频抽帧 + OCR + ASR 链路 | 技术 | Python 流水线 |
| AI 报告生成（6 维 + 修复建议） | 技术 | API + 模板 |
| H5 报告页 + 微信小程序前端 | 设计+技术 | 1.0 可用版 |
| 微信支付接入 | 技术 | 99/月 + 49 首月 |
| 用户协议、隐私政策、服务协议 | 法务 | 3 份文档上线 |

**Week 3（Day 15-21）：内测**
| 任务 | 负责人 | 交付物 |
|---|---|---|
| 内测客户招募（5 人，免费） | 创始人 | 5 个真实博主 |
| 内测反馈收集 | 创始人+顾问 | 反馈记录 + 改进清单 |
| Prompt 调优（基于反馈） | 内容 | Prompt v0.2 |
| Bug 修复 | 技术 | 0 P0 缺陷 |

**Week 4（Day 22-30）：正式发售**
| 任务 | 负责人 | 交付物 |
|---|---|---|
| 首批 5 篇案例内容生产 | 内容 | 公众号 + 视频 |
| 公众号矩阵搭建（3 个号） | 运营 | 公众号上线 |
| 抖音/视频号账号启动 | 运营 | 主理人账号 + 内容号 |
| 首场发售直播 | 创始人 | 引导首单 30 单 |

### 核心目标
- 验证投放 ROI > 1
- PRO 月增 100 单 / MAX 月增 30 单
- 月营收破 10 万 → 20 万

### 关键里程碑
- ✅ Day 120：付费投放 ROI > 1.2（投 1 元赚 1.2）
- ✅ Day 150：MAX 季度续费率 ≥ 70%
- ✅ Day 180：双产品总付费用户突破 1,000，月营收 20 万+

### 月任务拆解

**Month 4：投放启动**
- 抖音信息流投放（首月预算 3 万）
- 公众号广点通（首月预算 1 万）
- 小红书 KOC 矩阵（首月 20 个 KOC × 500 元）
- 投放素材 A/B 测试（10 套）

**Month 5：对标库扩张**
- 对标库扩到 20 赛道 × 10 = 200 头部

**Month 6：续费冲刺**
- 第一波 MAX 季卡到期续费
- 第一波 PRO 季付到期续费

- 启动 PRO 老用户升级 MAX 专项活动

## 8.7 高优先级风险与提前预案

| 风险 | 提前预案 |
|---|---|
| CT 引擎报告不准 | 内测 30 天严格验证，达不到 80% 满意度不上线 |
| MAX 客户首批不满意 | 创始人亲自参与首 20 单顾问会议 |
| 投放 ROI < 1 | 准备多套素材 + 多渠道 A/B 测试，设单月止损线 |
| 团队招不到顾问 | 提前 60 天启动招聘，找已有短视频背景的内容人 |
| 现金流断裂 | M3 启动天使轮，M9 启动 A 轮 |
| 平台政策变化 | 法务每月评估、多平台分散 |
