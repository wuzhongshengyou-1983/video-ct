## 第一：设计 Token 补全（基础设施，所有后续工作的前提）

### 1.1 新增 RGB 裸色分量 Token

**文件：** `apps/h5/src/styles/main.scss` — `:root` 块内

**当前缺失：** 全站 100+ 处 `rgba(245, 158, 11, X.XX)` 硬编码，分布在 10+ 种透明度。当前 Token 系统只有 `--vct-primary-dim`(0.12) 和 `--vct-primary-glow`(0.25)，无法覆盖其他透明度。

**需要新增：**

```scss
// RGB 裸色分量 — 用于 rgba(var(--vct-xxx-rgb), opacity) 动态透明度
--vct-primary-rgb: 245, 158, 11;
--vct-success-rgb: 48, 209, 88;
--vct-danger-rgb: 255, 69, 58;
--vct-warning-rgb: 255, 159, 10;
--vct-info-rgb: 100, 210, 255;
--vct-nebula-rgb: 96, 165, 250;
--vct-cosmic-rgb: 167, 139, 250;
--vct-accent-rgb: 251, 191, 36;
```

**收益：** 全局搜索 `rgba(245, 158, 11,` → `rgba(var(--vct-primary-rgb),` 即可，主色一变全站跟随。

---

### 1.2 新增缺失的 Surface 透明度档位

**当前缺失：** 设计系统只有 3 档 surface（0.04 / 0.08 / 0.12），但页面实际使用 0.02、0.03、0.06、0.10 等，无 Token 可匹配。

**需要新增 / 修正：**

```scss
--vct-surface-dim: rgba(255, 255, 255, 0.02);       // 极淡分隔
--vct-surface-subtle: rgba(255, 255, 255, 0.03);     // 微妙底板
--vct-surface: rgba(255, 255, 255, 0.04);            // 已有，不变
--vct-surface-hover: rgba(255, 255, 255, 0.06);      // ⚠️ 修正：原值为 0.08，应为 0.06
--vct-surface-raised: rgba(255, 255, 255, 0.08);     // 悬浮/激活（原 surface-hover 的值移到这里）
--vct-surface-active: rgba(255, 255, 255, 0.12);     // 已有，不变
```

**⚠️ 重要：`--vct-surface-hover` 从 0.08 修正为 0.06。** 上一轮 Token 统一中错误地将 0.06 背景映射到了 0.08 的 `--vct-surface-hover`，导致多处卡片「变亮」了 33%。原有 0.08 的使用场景迁移到新的 `--vct-surface-raised`。

**收益：** HistoryPage/GenePage/Growth.vue 中大量 `rgba(255,255,255,0.03)` / `0.02` 有 Token 可用。

---

### 1.3 新增边框宽度 Token

**当前缺失：** 全站 border 宽度在 0.5px、1px、1.5px、2px 之间随意混用，无规范。

**需要新增：**

```scss
--vct-border-width: 0.5px;        // 默认（玻璃卡片微边框）
--vct-border-width-md: 1px;       // 强调（输入框聚焦、激活卡片）
```

**收益：** 消除全站 border 宽度混乱。

## 第二：可访问性基线（全局性，优先执行）

### 2.1 `prefers-reduced-motion` — 一处改，全站受益

**文件：** `apps/h5/src/styles/main.scss`，在文件末尾新增

**当前缺失：** 全站所有动画（星空背景、骨架屏闪烁、抽屉滑入滑出、glow pulse、stagger 淡入等）对开启了「减弱动态效果」的用户照常播放。这是 WCAG 2.3.3 违规。

**改动：**

```scss
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
  .vct-skeleton,
  .vct-shimmer,
  .vct-glass-sweep {
    animation: none !important;
    background: var(--vct-surface) !important;
  }
}
```

**覆盖范围：** 所有 CSS 动画、Vue transition、骨架屏闪烁、星空背景。

## 第七：CelestialBackground 性能修复

**文件：** `apps/h5/src/components/CelestialBackground.vue`
